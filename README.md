<h1 style="text-align: center;">Simple Python Image & Metadata Viewer</h1>
<br>
  <p>A lightweight Python desktop application built with Tkinter and Pillow (PIL) that opens an image in a new window, resizes it cleanly to fit the screen, and displays both basic file information and hidden camera EXIF metadata right beside the image.</p>
<br>
<h2>--> Features</h2>
<p>* **Dual-Pane Layout:** Image on the Right, details neatly aligned on the left.<br>
* **Smart Resizing:** Uses Pillow's `thumbnail` feature to fit images seamlessly into a 500x500 window without stretching.<br>
* **EXIF Extraction:** Automatically decodes and displays advanced camera metadata (like Camera Make, Model, and Date/Time taken) for compatible images.</p>
